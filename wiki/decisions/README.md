# Decision Page Guide

Use ADR-style pages only for accepted decisions that constrain future work.
Proposals remain in `docs/` or `plans/` until accepted.

Recommended filename:

```text
NNNN-short-decision-name.md
```

Recommended sections:

1. Context
2. Decision
3. Consequences
4. Supersedes or superseded by
5. Sources

Do not delete superseded decisions. Mark them `historical` and link the newer
decision so agents can distinguish current direction from retained context.
