# Raw Sources

This directory is for immutable external source drops such as exported meeting
notes, third-party specifications, research captures, or supplied datasets.

Rules:

- Do not edit imported source contents.
- Use descriptive, date-stamped names.
- Store downloaded source assets under `raw/assets/` when needed.
- Register each useful source in [`../wiki/sources.md`](../wiki/sources.md).
- Do not copy existing `docs/`, `plans/`, code, Issues, or PRs here.

The repository's existing files and live GitHub state are already source
material and should be referenced in place.
