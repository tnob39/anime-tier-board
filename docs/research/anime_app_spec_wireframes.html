プロジェクト概要

目的

アニメの視聴管理 + Tierリスト作成・共有 + サブスクアフィリエイト（お小遣い程度）

ターゲット
アニメ好きの個人ユーザー
（特にTierリストを共有したい層）
収益モデル
U-NEXT / ABEMA / Disney+ / Amazon Prime のアフィリエイト

推奨技術スタック

Frontend Svelte + Vite（推奨）
Native Capacitor
DB Turso (SQLite + HTTP API)
AI Groq (Llama3) または Gemini Flash
Data manami-project + AniList API（選択的）

収益試算（現実的シナリオ）

主要判断事項

  • • manami-projectを初期データとして全件インポート（安全）
  • • AniListは最新情報取得のみに限定（規約対策）
  • • 画像は一旦グレー運用（AniList/MAL URL）→ 後でTMDB検討
  • • 最初は画像なし or 最小限でリリース → ユーザー反応を見て画像戦略再検討
  • • 配信情報は最初は手動登録（人気作品50〜100件）
